* [v] Результат залить в гит.


### Требования
* Python 3.6 (возможно, будет работать и на Python 3.4+).
* `pip install -r requirements.txt`


### Запуск
Выполнить следующую команду из папки `src`:
```
python3 manage.py runserver 127.0.0.1:8000
```
